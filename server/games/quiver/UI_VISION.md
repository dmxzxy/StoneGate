# QUIVER 界面愿景（UI_VISION）— 大胆版

> 不是"整理菜单"，是**换一个框架**：从「战斗画面 + 覆盖菜单」→「一座活着的猎人营地 + 一张世界远征图」。
> 风格：纯程序化 2D（火柴弓手 + 几何形 + 粒子/光照/视差），无美术资源——所以"大胆"靠**美术方向 + 演出 + 动效 + 信息编排**，不是贴图。
> 一句话：**营地是家，世界是猎场，菜单是营地里的站点，build 是你的弓手卡组。**

---

## 0. 美术方向（先立视觉身份，再谈布局）

**「暮色猎人 Dusk Hunter」**：暖篝火橙 × 冷暮色蓝的双调对比；扁平色块 + 柔和渐变 + 边缘光(rim light)；细线火柴弓手作主角剪影；纸纹噪点底；一个标志性强调色(琥珀金)。
- 昼夜循环 + 地区天际线视差背景 → 强烈"在某处"的临场感。
- 所有数字/图标走同一套排版令牌，**大数字只在该出现时出现**（预测 DPS、伤害、收益），平时克制。
- 目标气质：克制、精致、有呼吸感的挂机"家"，不是塞满进度条的仪表盘。

---

## 1. 五个大胆支柱

### 支柱一：营地 = 主界面（世界即 UI，diegetic）
主屏不再是"战斗 + 底部按钮"，而是**你的营地全景**。站点散布其中，点谁拉近到谁：

```
┌────────────────────────────────────────────┐
│ ☾ 暮色 · 晨曦绿野              ⚡许可4/5  🪙1840│  极简顶条：时段·所在地·资源
│        ☁           ☁              🏔        │  远景视差(当前地区天际线/昼夜)
│   🪵工坊                            🗺旗台    │
│      \              ✦              /         │
│       🔥         🧍弓手(穿你的装备,待机动画)    │  篝火 + 主角剪影(随装备变化)
│   ⚒铁砧                            📦箱柜     │
│  ══════════════ 营地地面 ════════════════════ │
│   ◀滑▶          [ 远 征 ▶ ]      Lv12 ▓▓▓▓░░ │  大主按钮"远征" + 经验
└────────────────────────────────────────────┘
   工坊=制箭三轴 · 铁砧=锻造 · 甲架=装备/精通 · 箱柜=背包 · 旗台=世界图
```
- **营地随成长进化**：帐篷→木屋→要塞；站点升级有可见变化 → 长线成就感（挂机游戏的"家"留存钩子）。
- **主角穿你的装备**：短弓/长弓/弩外形不同、护甲改剪影、附元素箭时箭尖带色 → 装备"看得见"。
- 副职业挂机时，弓手就在对应站点干活（在工坊制箭、在铁砧打铁、篝火边休息）——"看得见在干嘛"直接成立。

### 支柱二：世界远征图（地区/副本 = 一张旅程图，不是列表）
点"远征"或旗台 → 展开**世界路径图**（像 Roguelike 的节点旅程）：

```
┌────────────────────────────────────────────┐
│  世 界 远 征                              ✕  │
│   低级 ●━━━●━━━○━━━○━━━◇        你在这↑      │  走过=亮·当前=脉冲·未解锁=灰锁
│   晨曦  溪谷 荒原 森林  ◇兽穴(副本)            │  ◇=副本节点(肥掉落)
│            └─中级 ○━━○━━◇━━...━ 高级 ...    │  分档支线，越走越深
│   ┌──────────────────────────────────────┐ │
│   │ 幽暗森林  Lv7-10  推荐✓  熊/狼/食人魔    │ │  选中节点详情卡
│   │ 掉落: 桦木/铁矿  ·  [ 前往狩猎 ]         │ │
│   └──────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```
- 20 区 + 副本织成一条**可视化旅程**，有"走到哪了"的进度地图感，远胜下拉列表。
- 副本是图上的 ◇ 岔口节点，自然融入，不再"藏在地区里"。

### 支柱三：备战台 = 弓手卡组（build 像组卡，可拖、有协同灯）
甲架/工坊里有"备战"：把 build 做成**一眼看懂、可拖拽组合、协同会发光**的卡组面板：

```
┌────────────────────────────────────────────┐
│  备 战 · 弓手卡组                         ✕  │
│  弓 ▸ 「裂风」短弓   攻速 1.4s/次  ⚡快攻流派  │
│  箭袋 ▸ [钢簇·流血·风羽] [火箭·重羽] [ + ]    │  拖箭入袋槽，可带多种切换
│  技能轮 ▸ ➹强力  ⚡双重  ✚包扎  (按 prio)     │
│  ──────── 协同 ────────                      │
│  短弓 × 流血 ✦ +25%   ●亮                    │  契合点实时点亮/熄灭
│  火箭 × 不死 ✦ +20%   ○(当前地无不死,暗)      │
│  预测 DPS 342   ↑比上套 +18%                 │  换装即时预测，鼓励钻研
└────────────────────────────────────────────┘
```
- 把"快武器配某属性箭更好"变成**看得见、摸得着的发光协同**——build 感拉满，正中你要的"组合/build"。
- 多套弓袋预设可切（应对不同副本 boss 抗性）。

### 支柱四：战斗 = 横向 lane 演出（小型自走战，juicy）
战斗从"一对一回合"升级为**侧向推进的猎杀演出**：弓手在左，怪从右行进入场，箭矢抛物飞行、元素效果迸发、伤害跳字、击杀反馈、波次/boss 横幅。

```
┌────────────────────────────────────────────┐
│ 幽暗森林 · 夜                    ▓▓▓▓ 头狼boss│
│                          🔥(燃烧)            │
│  🧍─➹➹➹➹──────────🐺·····🐺······🐺          │  箭流 + 怪行进 + 元素特效
│  HP▓▓▓▓ MP▓▓░    残 3   wave 4/5            │
│  [➹3][⚡][⇶冷却][✚]  ← 技能冷却环固定屏底     │
└────────────────────────────────────────────┘
```
- 现有 ATB/技能/元素/抗性全用上，只是**演出更电影感**（入场/抛射/燃烧/中毒/暴击顿帧/击破爆裂）。
- 副本波次天然就是 lane 的一波波，体验连贯。

### 支柱五：明牌导航（大胆但零学习成本——所有入口看得见、带文字、双保险）
**砍掉手势**（学习成本高）。改为"营地站点 + 常驻底部 dock"双路径，怎么点都对：
- **营地站点都带文字标签**：图标下方一行字（🪵工坊·制箭 / ⚒铁砧·锻造 / 🛡甲架·角色 / 📦箱柜·背包 / 🗺旗台·远征），新手扫一眼就懂，老手直接点。
- **常驻底部 dock（带字，永远在）**：`[活动] [背包] [角色] [世界]` 四个明牌入口 —— 站点是沉浸快捷方式，dock 是保底，**任何功能至少两条可见路径能到，绝不藏**。
- 站点首次点亮有微光引导 + 一句提示；面板统一"标题 + 大 X + 带字 tab + 返回"，关闭方式永远一致。

```
营地主屏底部：    ┌─────┐┌─────┐┌─────┐┌─────┐
                 │ ▴   ││ 📦  ││ 🧍  ││ 🗺  │   ← 常驻 dock，图标+文字，明牌
                 │活动 ││背包 ││角色 ││世界 │
                 └─────┘└─────┘└─────┘└─────┘
```
- 原则：**视觉/演出大胆，操作明牌**。沉浸感来自营地/演出/卡组发光，不来自"藏起来让你找"。

## 2. 与现有系统/数据的映射（不是推倒，是重新装裱）
| 现有 | 在新框架里的家 |
|---|---|
| combat / 副本 | 支柱四 lane 演出（主舞台或"远征"进入） |
| gather / craft / forge / rest | 支柱一 营地站点（弓手在站点干活） |
| region / dungeon | 支柱二 世界远征图节点 |
| equip / 技能 / 精通 | 支柱三 备战台 + 甲架 |
| bag(72材料/三轴箭) | 箱柜 + 备战台弓袋 |
| 武器类型/元素箭/抗性/build | 支柱三 协同发光卡组（核心爽点） |
数据层(data/sys)几乎不动，**改的是 view 层的呈现与导航**——所以大胆但可落地。

## 3. 通往愿景的阶段路径（大胆目标，稳健落地）
> 仍走"组件层地基 + 逐区域"，但每区域照**愿景态**做，不是照旧样修补。每步绿灯门禁、独立 commit、可回退。

- **V0 美术方向 + 组件层**：定双调色/rim光/纸纹/令牌；`view/ui.lua` 通用件。先立"质感"。
- **V1 营地主屏**：营地全景 + 站点 + 主角穿装备 + 昼夜/视差 + "远征"主按钮（替换现底部按钮排为站点/手势）。
- **V2 世界远征图**：节点旅程图(地区+副本)。
- **V3 备战卡组**：弓袋槽 + 协同发光 + DPS 预测（build 爽点）。
- **V4 lane 战斗演出**：横向推进 + 元素/击杀演出升级。
- **V5 站点工作台**：工坊三轴制箭 / 铁砧锻造 / 甲架装备精通 / 箱柜背包，逐个精修。
- **V6 反馈与打磨**：顿帧/震屏/粒子/红点/昼夜过场，整体走查。

## 4. 需你拍板（方向先于细节）
1. **要不要走"营地+远征图"这套大重构？** 还是更想要"现框架但视觉/演出大幅升级"(保留战斗主屏+菜单，但按支柱三/四/美术方向做)？
2. **导航已定**：明牌——营地站点(带字) + 常驻底部 dock`[活动][背包][角色][世界]`，无手势、零猜测。（如想调 dock 的 4 个分组可说。）
3. **沉浸 vs 信息密度**：偏"克制电影感"(少 chrome、上下文显示) 还是"信息丰富仪表盘"(常驻多数据)？
4. **从哪个支柱先做出"哇"的一屏**：营地主屏 / 备战卡组 / lane 战斗——挑一个先做垂直切片给你看。

（仍是纯程序化 2D 火柴风，所有这些用形状+粒子+动效实现，不需要美术资源。**大胆在视觉与演出，明牌在操作。**）
