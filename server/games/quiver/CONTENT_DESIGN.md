# QUIVER 内容详细设计（CONTENT_DESIGN）

> 目标：把"基础可玩"扩成"能肝到 60 级满级、约 **100h+** 的完整内容"。
> 拍板：**等级上限 60 · 满级 ~100h(硬核) · 木/矿/草各 8 种独立材料 + 分级配方 · build 协同 ±20~30%**。
> 接现有锚点：`budget=GEAR_BUDGET*ilvl*rarity.mult*slot.w`；`1STR=+1攻击 / 1AGI=+0.6%攻速+0.04%暴击 / 1STA=+6生命`；
> 武器 `wmid*wspeed=budget*WEAPON_DPS_K`（DPS 守恒）；单发=攻击区间随机×箭档×暴击×(1-减伤)。
> 现有 20 区已铺好每档的 8 木/8 矿/8 草 kinds（见 data.REGIONS.nodes）—— 本设计把这些 kinds 从"展示名"升为**真实材料**，并在其上叠武器类型/元素箭/锻造/副本/build。

## 0. 内容地图与解锁节奏（体验曲线，~100h）

| 等级段 | 地区档 | 解锁的系统 | 目标累计时长 | 这段在玩什么 |
|---|---|---|---|---|
| 1–15  | 低级 6 区 | 采集细化、基础制箭、武器类型登场 | ~12h | 砍/挖/采分材料、铜铁箭、第一把蓝弓 |
| 16–35 | 中级 6 区 | **锻造(炼锭+造甲造弓)**、元素箭、副职业等级压力 | ~33h | 建第一套自锻装、试元素箭 build |
| 36–55 | 高级 8 区 | **副本**、命名传说武器、build 成型 | ~70h | 刷副本攒传说/特殊材料、定型 build |
| 56–60 | 王座区 + 顶级副本 | 满级冲刺、毕业装、**战斗精通(软成长)** | ~100h+ | 极限 build、传说毕业、精通无限成长 |

**经验曲线**：`xp_need(L)=floor(55*L^2.15)`（比现 `80*L^1.6` 更陡，把后段拉长）。
- 累计到 60 级 ≈ **1.0~1.1e7 xp**。配合下面"每源 xp"，战斗为主、副本翻倍效率，肝满 ≈100h。
- 战斗 xp：`floor(enemy_level*4)`（去掉 +10 平铺，让高区更值）；精英 ×2、稀有 ×4。
- 副本结算 xp：`boss_level*60`（约等于一波小怪，但一次性给 + 掉落更肥 → 后期主力 xp 源）。
- 采集/制造 xp 只喂副职业等级（不喂角色等级，避免挂采集白嫖满级）。
- **满级后**：xp 转「战斗精通」点（每 `xp_need(60)` 一点），精通点投入小幅永久加成（攻击/暴击/采集速等，每点 +0.5%，软上限靠成本递增）—— 给 100h 后仍有目标，但不强制、不破坏平衡。
> 注：曲线/每源 xp 是**可调初值**，需落地后用无头模拟+实机校准到 ~100h；先给公式与配比框架。

## 1. 角色属性成长 / 经验 / 等级上限

- **上限 60**。到 60 停止升级，xp 转精通（见上）。
- **每级**：`+2 力量 +2 敏捷 +3 耐力`（沿用）。60 级累计基础 ≈ 力123/敏123/耐182 → `max_hp≈60+182*6=1152`，`attack 基础≈5+123`，`crit≈0.05+123*0.0004≈0.099`（其余靠装备/箭/技能/暴击词缀堆）。
- **属性来源**：等级(自动) + 装备(主属性/词缀) + 技能 buff(限时)。不做手动加点（保持挂机轻操作；如果想要加点是另一个决策）。
- **MP**：`max_mp=30+lvl*5`→60 级 330，回 6/s（沿用）。后期高费技能轮转靠它节流。

## 2. 武器类型 + 箭矢类型（build 核心）

### 2.1 武器三类型（差异轴 = 攻速带 + 内置暴击，DPS 守恒）
装备槽 `bow` 泛化为 `weapon`，gear 新增 `wtype`。三类型都走 `wmid*wspeed=budget*WEAPON_DPS_K`，**裸 DPS 相等**，差异在节奏与 build 契合：

```lua
D.WEAPON_TYPES = {
  shortbow = { name="短弓", spd={0.72,0.92}, crit=-0.02, tag="快攻：每击触发多 → 利流血/毒/冰减速叠加" },
  longbow  = { name="长弓", spd={0.50,0.64}, crit= 0.00, tag="均衡通用：元素 DOT 覆盖稳" },
  crossbow = { name="弩",   spd={0.34,0.46}, crit=+0.06, tag="重击高暴：利大伤技能/穿甲重箭" },
}
```
- roll_gear(weapon)：随机(或地区偏向)选 wtype；`wspeed=rand(spd[1],spd[2])`，`wmid=budget*WEAPON_DPS_K/wspeed`（保持守恒）；`crit_innate=wtype.crit`（写进 gear.stats，recalc 计入）。
- 展示：`攻击 wmin~wmax · 攻速 X.XX 秒/次 · [短弓/长弓/弩]`。

### 2.2 稀有度命名规则（你的要求：蓝色以上唯一名）
- **粗糙/普通/优秀(白绿)**：程序名 = `品质 + 材料前缀 + 类型`（如「优秀 精铁短弓」），随机词缀(0~1 条)。
- **精良+(蓝/紫/橙)**：必须**唯一名 + 签名特效**。数据驱动：`D.NAMED_WEAPONS`，每条 = {name, wtype, min_ilvl, signature(固定特效/高词缀), flavor}；roll 蓝+武器时从"类型匹配 且 ilvl 够"的池里抽一个未拥有的，叠加随机词缀。
```lua
-- 样例（落地补全到每档每类型 3~5 个）
D.NAMED_WEAPONS = {
 { name="裂风",   wtype="shortbow", min_ilvl=10, sig={haste=true, bleed_on_hit=0.08}, flavor="出手如风，箭箭见血" },
 { name="贯石弩", wtype="crossbow", min_ilvl=14, sig={armor_pierce=0.25, crit=0.08},  flavor="一矢洞穿顽石" },
 { name="林语长弓",wtype="longbow", min_ilvl=12, sig={ele_amp_fire=0.2},               flavor="附火之箭灼焰更盛" },
 { name="霜噬",   wtype="shortbow", min_ilvl=30, sig={chill_amp=0.3, haste=true},      flavor="" },
 { name="王座裁决",wtype="crossbow",min_ilvl=55, sig={armor_pierce=0.4, crit=0.15, big_hit=0.25}, flavor="毕业级" },
}
```
- 这样蓝+武器永远有名有签名特效，且特效直接喂 build（如「裂风」短弓自带流血叠加，配流血箭爆发）。

### 2.3 箭矢类型（核心内容：三轴组合 = 箭头档 × 元素/特效 × 翎羽；±20~30% build）
箭是消耗品也是 build 的一半，**三轴自由组合**，组合数远大于单表，制箭因此是长线追求：

**① 箭头档（决定物理伤害倍率，吃"箭簇矿"锭，10 档）**
| 档 | 燧石 | 铜簇 | 青铜簇 | 铁簇 | 钢簇 | 银簇 | 秘银簇 | 精金簇 | 星钢簇 | 虚空簇 |
|---|---|---|---|---|---|---|---|---|---|---|
| 倍率 | 0.9 | 1.1 | 1.3 | 1.55 | 1.85 | 2.15 | 2.5 | 2.85 | 3.2 | 3.6 |

箭杆木需 ≥ 箭头档（不够则脆，批量减半），鼓励木/矿同步推进。

**② 元素/特效（决定 on-hit 玩法，吃"精萃草"/怪物掉落；10 种）**
| 箭 | 关键材料 | 效果 | 最契合 |
|---|---|---|---|
| 物理 | （无附加） | 纯物理，吃满暴击 | 弩(高暴)、破甲 build |
| 火焰 | 油(怪掉) | 点燃 DOT 0.3×/s×4s，可叠新覆旧 | 长弓(稳覆盖)·克不死 +20% |
| 冰霜 | 精萃草(冰系) | 减速：敌出手速 -20% 3s | 短弓(高频刷新)·风筝 |
| 剧毒 | 毒囊(虫掉)/毒性草 | 中毒叠层，每层 0.06×/s（命中+1层，上限10） | 短弓(叠层快) |
| 流血 | 利刃石(矿副产) | 物理流血叠层，**无视护甲** | 短弓+「裂风」·对高甲 |
| 穿甲 | 重锭 | 无视 30% 护甲，单发更重、略降速 | 弩+大伤技能·对石巨人/重甲 |
| 雷击 | 雷击木+精萃 | 命中有概率连跳 2 目标(为群战预留) | 短弓·副本多目标 |
| 爆裂 | 油+硫(矿) | 命中小范围溅射 0.4×(群战/波次) | 副本清波次 |
| 破甲 | 精萃草(蚀系) | 命中降敌护甲 8%/层(可叠)，团队向自叠 | 弩/长弓 持续 |
| 净化 | 月华草 | 对不死/虚空 +35% 伤，对其它无加成 | 高区不死/虚空副本 |

**③ 翎羽（决定手感微调，吃"羽毛"变体；4 种，小幅 build 杠杆）**
- 普通羽：无修正。鹰羽：+暴击率 3~6%（配弩/物理）。风羽：+攻速 4~8%（配短弓叠层）。重羽：+穿透/稳定，单发 +5% 但 -3% 速（配长弓/穿甲）。

> 一支箭 = 箭头档(伤害) + 元素(玩法) + 翎羽(微调)。例：**钢簇·流血·风羽** 配短弓「裂风」= 流血暴叠核心箭。
> 高端可造**签名箭**（稀有配方解锁，自带组合特效+更高数值），如「龙息箭」(火+爆裂)、「弑神箭」(穿甲+净化+星钢簇) 作毕业目标。

- **build 数学(±20~30%，乘算封顶 +30%)**：
  - 短弓(快) × 叠层箭(流血/毒/冰) + 风羽：单位时间命中多 → 叠层快 → 对耐久怪 **+~25%**。
  - 弩(慢重) × 穿甲/破甲 + 大伤技能(强力/瞄准) + 鹰羽：单发大且破甲高暴 → 对高甲 **+~25%**。
  - 长弓 × 火/净化：DOT 覆盖稳，对不死 **+~20%**。
  - 元素 vs 怪物抗性(见 §4)再 ±20%（弱点/抵抗）。封顶防膨胀。

实现：箭表拆成 `ARROW_HEADS`(档→phys_mult) / `ARROW_ELEMENTS`(element→proc 数据) / `ARROW_FEATHERS`(微调)；一支成品箭 = 三者 id 组合 + 派生显示名/颜色；战斗 do_shot 命中按 element 挂 dot/debuff、按 feather 改暴击/速、按 family 抗性缩放。

实现：箭表加 `phys_mult / element / proc`；战斗 do_shot 命中后按 element 挂 dot/debuff；敌按 family 改 element 系数；wtype 改 proc 频率/数值。

## 3. 资源细化（每档 3 种 = 木/矿/草 各 24 种 + 二级材料）

### 3.1 主材料：8 档 × **每档 3 种功能变体**（每种偏一条制造线，gather 多样性 + 配方深度）
每档 3 个变体不是换皮，而是各**偏向一条制造线**，让"采什么"有取舍、配方有讲究：
- 木材三系 → **箭杆木**(轻直，制箭杆)、**弓臂木**(韧弹，造弓)、**薪炭木**(耐烧，炼锭燃料/护甲衬)。
- 矿石三系 → **箭簇矿**(硬利，造箭头)、**兵刃矿**(锋锐，造武器)、**甲胄矿**(坚韧，造护甲)。
- 草药三系 → **疗愈草**(回复药剂)、**精萃草**(元素/buff 药)、**毒性草**(毒/debuff 与毒箭)。

| 档 | 箭杆木 / 弓臂木 / 薪炭木 | 箭簇矿 / 兵刃矿 / 甲胄矿 | 疗愈草 / 精萃草 / 毒性草 | 区段 |
|---|---|---|---|---|
| T1 | 橡木 / 柳木 / 松木 | 燧石 / 铜矿 / 锡矿 | 三叶草 / 微光花 / 毒芹 | L1-7 |
| T2 | 桦木 / 榆木 / 杉木 | 赤铜 / 青铜砂 / 铅矿 | 薄荷 / 萤草 / 颠茄 | L6-12 |
| T3 | 梣木 / 槐木 / 桤木 | 铁矿 / 磁石 / 灰铁 | 鼠尾草 / 星彩花 / 乌头 | L11-19 |
| T4 | 紫杉 / 山核桃 / 雪松 | 银矿 / 钴矿 / 镍矿 | 夜帽菇 / 月华草 / 鸩羽叶 | L18-28 |
| T5 | 铁木 / 龙骨木 / 焦木 | 秘银 / 精钢砂 / 玄铁 | 炽焰花 / 灵泉草 / 蚀骨藤 | L26-36 |
| T6 | 黑栎 / 影桦 / 炭橡 | 精金 / 寒钢 / 厚铸石 | 霜百合 / 龙息兰 / 腐心花 | L34-46 |
| T7 | 炽炎木 / 雷击木 / 黯杉 | 星钢 / 烈焰岩 / 玄武铸 | 曼德拉 / 星髓花 / 蚀魂菇 | L44-55 |
| T8 | 世界根 / 永恒藤 / 虚影木 | 虚空铁 / 噬星矿 / 神化晶 | 虚空花 / 创世露 / 湮灭孢 | L54-60 |

- 采集节点产**该地区对应档+随机一系**的材料；想专精某线（如一直采"箭簇矿"做箭）就反复挑对应节点。
- 配方按系取材：制箭=箭杆木+箭簇矿(+羽毛/精萃)；造弓=弓臂木+兵刃矿；造甲=甲胄矿+皮革；炼锭=矿+薪炭木(燃料)。
- 实现：`MAT_NAME/MAT_COLOR/MAT_DESC` 补全 72 项；`icon_mat` 仍按木/矿/草大类画形、按档调色、按系加小标记(箭簇=尖角/兵刃=刃纹/甲胄=块面)。data.REGIONS.nodes.kinds 由"单名"扩成"该档三系"。

### 3.2 二级 / 特殊材料
- **羽毛 feather**：所有箭必需。来源：野兽怪掉(boar/wolf 30%) + 砍柴小概率(鸟巢 10%)。低门槛、稳定流通。
- **油 oil**：火箭用。来源：野兽/食人魔掉(15%)。
- **毒囊 / 霜露 / 利刃石**：元素箭附材，分别来自虫系怪掉 / 冰区草 / 矿区副产。
- **兽皮 hide → 皮革 leather**：护甲链（hide 怪掉 → forge 鞣制成 leather）。
- **锭 ingot**（见 §5 锻造）：copper_ingot…voidiron_ingot，造箭头/武器/护甲。

> 经济原则：采集给主材 + 羽毛/皮；战斗给油/毒囊/皮/装备；锻造把矿→锭→装备/箭头。三态(采集/战斗/制造锻造)互相喂，鼓励轮换。

## 4. 怪物种类 + 地区分布 + 抗性（喂 build）

### 4.1 怪物家族（新增 family + 元素抗性，让箭元素有意义）
现有 10 种归family，并补几种到每档不重样：
```lua
-- family: beast/humanoid/undead/construct/elemental/dragon
-- resist: 各元素系数（>1 弱点，<1 抵抗，=1 普通）。掉落 drops 指定特殊材料。
D.ENEMY_FAMILY = {
  beast    = { resist={fire=1.1, frost=1.0, poison=0.9}, drops={feather=0.3, oil=0.15} },
  humanoid = { resist={fire=1.0, frost=1.0, poison=1.0}, drops={hide=0.2} },
  undead   = { resist={fire=1.25,frost=0.8, poison=0.7}, drops={oil=0.1} },     -- 怕火
  construct= { resist={fire=0.9, frost=0.9, poison=0.5}, armor_mul=1.3,         -- 高甲，怕穿甲
               drops={ore_bonus=true} },
  elemental= { resist="self",  drops={essence=0.2} },   -- 抵抗同系、弱反系
  dragon   = { resist={fire=0.7,frost=0.7}, drops={oil=0.4, scale=0.3} },        -- boss 系
}
```
- 新增敌型铺满每档（示例）：虫(虫系→毒囊)、巨蝠、石像鬼、冰狼、熔岩兽、虚空使徒、亡灵法师…（沿 ENEMY_ARCH 模板补 hp/dmg/armor/spd/color + family）。
- 战斗：`element_dmg *= family.resist[element]`；construct `armor*=1.3`（逼穿甲/弩 build）。
- 地区分布：沿用 `region.enemies`，按 family 主题化（湿地多虫/不死、熔狱多元素/龙）。

## 5. 锻造系统（新增：炼锭 + 造甲造弓）

新增**锻造**子职业活动 `forge`（group=sub，与采集/制造并列），有独立等级 `forge.lvl/xp`（做工攒、解锁更高配方）。

### 5.1 两类锻造配方
1. **炼锭**（矿→锭，stackable）：`copper×3 → copper_ingot`，铁/银/秘银/精金/星钢/虚空铁同理；高档需燃料(煤/油)与更高 forge 等级。锭用于箭头、武器、护甲。
2. **造装**（产出 **gear**，带稀有度 roll）：
```lua
-- out.kind="gear"：do_craft 调 roll_gear(slot, ilvl, rarity)
-- ilvl 由材料档决定；rarity 由 forge 等级 + 材料 + 少量随机(可加"精工"消耗品提升蓝+概率)
{ id="iron_chest", name="精铁胸甲", kind="forge", req_forge=4, slot="chest",
  cost={ iron_ingot=4, leather=2 }, ilvl_base=12, rarity_roll={common=0.5,uncommon=0.35,rare=0.15} },
{ id="iron_longbow", name="精铁长弓", kind="forge", req_forge=5, slot="weapon", wtype="longbow",
  cost={ iron_ingot=5, ash=3 }, ilvl_base=14, rarity_roll={uncommon=0.5,rare=0.4,epic=0.1} },
```
- **造装是 drop 之外的第二条出装线**：可定向做某槽/某武器类型（drop 随机，锻造定向）→ build 成型靠锻造补齐缺口件。
- 蓝+造出也走 §2.2 命名（造出「裂风」之类需稀有图纸/材料，做"配方解锁"长线目标）。

### 5.2 UI：制造页加"锻造"分类 tab（炼锭 / 造甲 / 造弓），复用图谱列表+持续制作直到缺料。

## 6. 副本系统（新增：后期主循环，撑 36–60 / 100h 尾段）

### 6.1 结构
- `D.DUNGEONS`：每档若干副本，按地区进度解锁。一个副本 = **固定波次(3~5 波小怪) + 1 boss**，自动战斗用你当前 build 打。
- **进入成本**：消耗"探险许可"(energy，随时间恢复，封顶) 或 boss 钥匙(掉落)。挂机友好：许可恢复 = 离线也涨，鼓励定期回来开副本（典型挂机留存设计）。
- **结算**：经验大包(`boss_level*60`) + **更肥掉落表**（保底 rare+、unique 武器/特殊材料/精工/钥匙）。失败(阵亡)则减少损失、给少量安慰产出。

```lua
{ id="darkwood_warren", name="幽林兽穴", tier="low", min_lvl=8, waves=3, boss="alpha_wolf",
  unlock="darkwood", cost_energy=20, drops={ rar_floor="rare", unique_chance=0.05, mats={oil=3,hide=4} } },
{ id="cinder_forge",   name="熔狱熔炉", tier="high",min_lvl=45,waves=5, boss="lava_tyrant",
  unlock="cinder",     cost_energy=40, drops={ rar_floor="epic", unique_chance=0.18, mats={starsteel=4, essence=2}, key_chance=0.1 } },
```
- **boss 机制喂 build**：高甲 boss(逼穿甲/弩)、高频小怪 boss(逼 AOE/叠层短弓)、元素 boss(逼对应元素抗性箭)。→ 不同副本奖励不同 build，鼓励多套切换。

### 6.2 UI：底部入口或地区页内"副本"分页；选副本→看推荐等级/掉落→开打→波次进度条+boss 战→结算弹窗。

## 9. 需要你拍板的剩余开放项（带推荐）
1. **手动加点？** 现为全自动属性成长。推荐**保持自动**（挂机轻操作）；要手动加点(力/敏/耐自选)是另做。
2. **副本进入成本** = 时间恢复的"探险许可"(推荐，挂机留存) vs 纯钥匙掉落 vs 无限制可刷？推荐**许可+boss 钥匙**双轨。
3. **满级软成长** = 战斗精通点(推荐) vs 直接停止成长 vs 装备无限附魔？推荐**精通点**，温和无限、不破平衡。
4. **命名武器获取** = drop + 锻造解锁配方都能出(推荐) vs 仅副本 boss 掉？推荐**双途径**，锻造给定向、副本给惊喜。
